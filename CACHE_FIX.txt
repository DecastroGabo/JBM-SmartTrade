2. If that doesn't work, CLEAR BROWSER CACHE:

   Chrome/Edge:
   - Press Ctrl + Shift + Delete
   - Select "Cached images and files"

   - Click "Clear data"

   - Reopen browser
   - Go to: http://localhost/jbmtrading/

-------------------------------------------------------------------------

3. If still not working, try INCOGNITO MODE:

   - Press Ctrl + Shift + N (Chrome/Edge)
   - Visit: http://localhost/jbmtrading/
   - If it looks correct here, your regular browser cache needs clearing

-------------------------------------------------------------------------

WHY THIS HAPPENS:

Your browser saved the old CSS file and keeps showing it even though
the actual CSS file is correct. This is called "caching."

I've updated your files to force fresh CSS/JS loading every time, so
after you clear your cache once, this won't happen again!
